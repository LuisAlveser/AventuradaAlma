📌 Sobre
O Aventura da Alma  é uma plataforma SaaS que utiliza IA generativa (Google Gemini) para criar histórias completamente personalizadas baseadas no hiperfoco de cada criança com TEA. A história é gerada considerando o nome da criança, seus animais de estimação, amigos, familiares, nível de alfabetização e nível do espectro autista — tornando cada leitura uma experiência única e terapeuticamente relevante.

✨ Funcionalidades

🤖 Geração de histórias com IA — histórias únicas baseadas no perfil de cada criança
👧 Perfil da criança — cadastro com hiperfoco, nível de autismo, alfabetização, amigos e família
💾 Biblioteca de histórias — salve e releia as histórias favoritas
💳 Pacotes com benefícios  — integração completa com Stripe
🔐 Autenticação segura — JWT
📱 Design responsivo — funciona em mobile e desktop
